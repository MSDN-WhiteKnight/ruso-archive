Печать Word файла с диалоговым окном "печать" с помощью C#

Source - by Bumable

Нужно по нажатии кнопки открыть диалоговое окно для печати word документа. Доступен метод word.printout();, но при его вызове ничего не происходит. Я вообще мало понимаю в печати из word приложения.

Answer 837457

Source - by MSDN.WhiteKnight

Как-то так:

using Word = Microsoft.Office.Interop.Word;
...

private void button1_Click(object sender, EventArgs e)
{
    Word.Application wordApp = new Word.Application();
    wordApp.Visible = true;            
    string file = "c:\\test\\test.doc";

    var docs = wordApp.Documents;
    var doc = docs.Open(FileName:file);
    doc.Activate();

    int dialogResult = wordApp.Dialogs[Word.WdWordDialog.wdDialogFilePrint].Show();
}
