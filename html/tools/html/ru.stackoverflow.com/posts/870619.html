Почему не работает код на с# для вывода бинарного ресурса?

Source - by Коновалов Максим

File.WriteAllBytes(@"C:\Users\1\Desktop\123\smth.cs", WindowsFormsApp12.Properties.Resources.smth);

Ошибка : resources не содержит определения для smth (подчеркивает smth)

Суть : по нажатию на кнопку нужно вывести в папку 123 на рабочем столе файл smth.cs из ресурсов

Answer 870942

Source - by MSDN.WhiteKnight

Класс Properties.Resources содержит автосгенерированный код для доступа к ресурсам проекта, каждому файлу ресурса соответствует свойство в этом классе. Для того, чтобы там появился "smth", нужно добавить одноименный ресурс в свойствах проекта на вкладке "Ресурсы".
