Answer 10249

Source - by aleksandr barakin

алгоритм поиска:

  1. на каждой странице сайта есть ссылка «Метки».
  2. в поле поиска вписываете интересующую вас тему («битрикс») и просматриваете все найденные метки, начиная с «битрикс».
  3. на таких страницах есть ссылка «Лучшие участники».
  4. просматриваете профили участников. там есть список лучших сообщений (по центру) и (возможно) контактная информация, которой участник захотел поделиться (вверху справа).
