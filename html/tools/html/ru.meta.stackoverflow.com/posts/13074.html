Открываем справку:

Ответы, которые совершенно не отвечают на вопрос, могут быть удалены. Это относится к ответам, которые:

  • являются комментарием к вопросу или ответом на совсем другой вопрос;
  • ...
  • даже частично не отвечают на фактический вопрос.

Answer 13075

Source - by Anton Menshov

Картинка взята из широко известного в узких кругах ответа на вопрос от том, "что есть not an answer" c Meta Stack Overflow.

Ключевые моменты из этого же поста с моим вольным переводом:

You flag as NAA things posted as answers that clearly and obviously do not attempt to answer an on-topic question or the question asked.

Использовать флаг "не является ответом" нужно на ответы, которые ясно и очевидно не пытаются отвечать на поставленный вопрос (который подходит по тематике сообщества).

В переводе я выделил "ясно и очевидно". Очевидно и ясно должно быть не только человеку, который ставит флаг, но и любому модератору, который может понятия не иметь о конкретной технологии, использующейся в вопросе.

Также:

Any post that attempts to answer the question — however badly — is still an answer! Do not use the "not an answer" flag for wrong answers or answers that use a different programming language. Moderators do not judge the technical correctness of answers.

Любой пост, который пытается ответить на вопрос, неважно насколько плохо, все равно является ответом. Не используйте флаг "не является ответом" для неправильных ответов или ответов, которые используют другой язык программирования. Модераторы не оценивают техническую корректность ответа.

Исходя из этого, ваш флаг нужно отклонять. Правильное действие (если вы считаете ответ плохим) - "голос вниз".

Краткая мотивация:

  • нагрузка на модераторов будет огромной, если начать ставить такие флаги.
  • "ясно и очевидно" - должно трактоваться с максимально строгой стороны. Модератор не должен обладать НИКАКОЙ технической экспертизой по вопросы, чтобы принять решение об утверждении флага.
  • для таких ситуаций уже есть кнопка "голования против".
