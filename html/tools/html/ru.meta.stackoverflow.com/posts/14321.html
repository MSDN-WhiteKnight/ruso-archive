Существует ли отдельное приложение Stack Overflow на русском?

Source - by Matvj

Существует ли приложения для Stack Overflow на русском для ПК(желательно под Linux) и для мобильных устройств(желательно под iOS)?

Answer 14322

Source - by αλεχολυτ

Приложений непосредственно для "Stack Overflow на русском" никогда не было. Но были приложения для доступа к различным сайтам сети StackExchange. В какой-то момент было даже выделенное приложение для англоязычного Stack Overflow.

На текущий момент все приложения признаны устаревшими и определённые API отключены. То есть, скорее всего, они даже не позволяют просматривать контент. Я лично раньше пользовался приложениями под iOS и под Android.

На MSE достаточно большое кол-во вопросов по приложениям можно найти по меткам "ios-app" и "android-app".
