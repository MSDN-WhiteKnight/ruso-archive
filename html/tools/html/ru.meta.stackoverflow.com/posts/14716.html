Будет ли метка автоматически удалена, если по ней больше нет вопросов, но кто-то её отслеживает?

Source - by Vitalizzare

Допустим, кто-то задал вопрос с меткой abc. Второй пользователь перешел на описание метки и нажал на кнопку Отслеживать. После этого третий пользователь удалил эту метку из единственного вопроса, посчитав её лишней. В итоге имеем метку без вопросов, но с пользователем, который её отслеживает. Будет ли эта метка автоматически удалена?
