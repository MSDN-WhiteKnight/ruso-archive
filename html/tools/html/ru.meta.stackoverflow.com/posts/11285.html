Как сделать, чтобы код в ответах редактировался сам?

Source - by Максим Фисман

Часто (почти всегда), давая ответ на вопрос, код пишется сразу, то есть не где-то в редакторе или IDE, а сразу вручную между тройных апострофов ```.

Возможно ли в таком случае сделать так, чтобы пробелы при открытии новой скобки ставились сами? Например, я пишу:

public class myClass {
    public void myMethod () {
        // здесь на каждой строке нужно ставить по 8 пробелов 
    }
}

Eсть ли способ, чтобы эти табуляции проставлялись сами после открытой фигурной скобки или там после двоеточия в питоне и т.п.?
