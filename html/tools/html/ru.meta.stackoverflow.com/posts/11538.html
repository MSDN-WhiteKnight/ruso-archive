По моему скромному мнению метка поздравление актуальна для сообщений, адресованных какому-то конкретному участнику (или может быть небольшой группе, хотя пока сложно придумать для этого пример). Если же сообщение описывает важную контрольную точку для всего сообщества, то здесь уместна метка вехи. В некоторых ситуациях их разумно использовать совместно, например:

  • первый участник, набравший 100, 200, 500, 1000 килореп
  • первый участник, получивший золотые метки по N популярным меткам
  • первый участник, сообщение которого набрало 100, 200, 500 ... голосов "за"
  • ...
