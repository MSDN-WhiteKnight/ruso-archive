Нужно сжечь [Мозиллу]

Source - by insolor

Есть (была, пока я не удалил ее из всех 60+ вопросов) метка mozilla, которую беспорядочно ставили вместе или вместо firefox, thunderbird, gecko, и в принципе теоретически ее могут использовать для любого из продуктов Mozilla.

Есть также недавнее (принятое!) предложение правки краткого описания метки, где предлагается использовать эту метку для обозначения Mozilla Firefox (при живой метке firefox).

Мое предложение:

  • Сжечь (удалить окончательно и запретить создавать) метку mozilla
  • Для firefox добавить синоним mozilla-firefox (именно для этой метки, т.к. mozilla в большинстве случаев использовалась как синоним firefox)
