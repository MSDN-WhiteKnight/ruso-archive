Все аватарки по приведённым ссылкам - из гугла. Засим возможны 3 варианта:

  1. Что-то сломалось в части авторизации на Stackoverflow (маловероятно)
  2. Что-то сломалось у гугла в части отдачи аватарок юзеров (вероятно)
  3. В API гугла что-то поменялось в части отдачи аватарок юзеров без поддержки обратной совместимости (вероятно)
