Source - by Зонтик

Синонимизировать не надо. Надо разобрать беспорядок.

  • Для вопросов про операционные системы есть метка ос с неоформленными синонимами, либо метки windows, linux и так далее.

  • Для вопросов про файловую систему есть отдельная метка (файловая-система).

  • Для вопросов про класс System в разных языках можно создать отдельные метки для каждого языка, либо оставить system, но добавить описание. Пока создал java.lang.system. Предложил несколько правок вопросов с метками javasystem с заменой system на java.lang.system (но правки улучшали и тела вопросов!).
