Перенос вопросов с Meta на основной сайт

Source - by Dev18

Нашла похожий вопрос: Перенос вопросов с Меты, но в нём нет ответа.

При закрытии вопроса на основном сайте доступна причина "принадлежит Meta", и в этом случае вопрос может быть перенесён на соответствующий Meta-сайт.

Обратного механизма я не обнаружила:

если на Meta по ошибке задают вопрос по программированию (который должен быть размещён на основном сайте), среди причин закрытия нет варианта переноса на основной сайт.

Под указанным вопросом есть комментарий:

У модераторов на этом диалоге есть выбиралка для любого сайта из сети. Если надо - оставляйте тревоги

Правильно ли я понимаю, что в такой ситуации единственный корректный порядок действий для обычных участников - оставить тревогу (флаг) для модератора с просьбой о переносе?

В связи с этим уточняю:

  • Перенос с Meta на основной сайт доступен только модераторам?
  • Является ли отсутствие обратной миграции намеренным решением?

Хотелось бы понять, существует ли иной предусмотренный механизм или действительно следует просто оставлять тревогу.
